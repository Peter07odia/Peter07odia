### 🚀 Featured Projects

#### 🏥 AI Customer Service Agent — Walk-In Clinic
An end-to-end automated customer service system built for a walk-in clinic. Patients can book and manage appointments through a conversational AI agent, while internal staff workflows are automated to significantly reduce manual workload.
- **Stack:** AWS Lambda · AWS Bedrock · S3 · Claude (via Claude Code) · API integrations
- **Outcome:** Measurable reduction in staff time spent on repetitive administrative tasks

#### 👗 Glook (Colors of Life) — AI Fashion App
A full-stack production mobile app where users create photorealistic AI avatars and virtually try on clothing. Features **Emma**, a GPT-4o AI personal stylist with RAG over the user's wardrobe, multi-provider fashion search (Brave + Google Shopping), weekly outfit planning, weather-aware recommendations, Apple IAP, and Google Veo video generation — orchestrated through n8n agentic workflows and Supabase edge functions.
- **Stack:** React Native (Expo SDK 54) · TypeScript · Supabase · fal.ai · OpenAI GPT-4o · Gemini · Google Veo · n8n · Brave Search · GitHub Actions · EAS
- **Outcome:** 30% increase in user engagement; shipped avatar creation, virtual try-on, AI stylist, smart wardrobe, and outfit planner

#### 🎬 NVIDIA Video Summarization Agent
An agentic pipeline that ingests video content and extracts structured insights for rapid review.
- **Stack:** n8n · NVIDIA NIM · Python
- **Outcome:** 70% reduction in manual video review time

#### 🏠 Homify — AI Interior Design Tool
AI-assisted concept visualization tool combining GPT-4o prompting with inpainting workflows for rapid interior design ideation.
- **Stack:** GPT-4o · Stable Diffusion inpainting · Python
